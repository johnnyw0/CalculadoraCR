#  Calculadora de CR

Este projeto é uma aplicação desktop para calcular o Coeficiente de Rendimento acadêmico.

## Fórmula de Cálculo

### UFF
O CR é calculado utilizando a média ponderada das notas, onde o peso de cada nota é a carga horária da respectiva matéria. A fórmula matemática é a seguinte:

$$ CR = \frac{\sum_{i=1}^{N} (n_i \cdot c_i)}{\sum_{i=1}^{N} c_i} $$

Onde:
- $N$ é o número total de matérias.
- $n_i$ é a nota da i-ésima matéria.
- $c_i$ é a carga horária da i-ésima matéria.
